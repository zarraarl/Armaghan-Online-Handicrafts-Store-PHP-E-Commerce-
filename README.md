## üöÄ Quick Start

1.  **Clone the repository** to your local server directory (e.g., `htdocs` or `www`).
2.  **Database Setup:**
    * Create a MySQL database named `sonicTechStore`.
    * Import the provided SQL file (or create `Customers` and `Products` tables).

    * Open `dbconnect.php`.

4.  **Run:**
    * Start your Apache/MySQL server (XAMPP/WAMP).
    * Navigate to `http://localhost/your-folder-name/home.php`.
